Tag - h1 até h6

São utilizadas para definir cabeçalhos ou títulos, sendo h1 o mais importante e h6 o menos importante.

Exemplo: 

        
          <h1>Exemplo de tag h1</h1>
          <h2>Exemplo de tag h2</h2>
          <h3>Exemplo de tag h3</h3>
          <h4>Exemplo de tag h4</h4>
          <h5>Exemplo de tag h5</h5>
          <h6>Exemplo de tag h6</h6>

Resultado abaixo:

Exemplo de tag h1

Exemplo de tag h2

Exemplo de tag h3

Exemplo de tag h4

Exemplo de tag h5
Exemplo de tag h6

Tag - p

Representa um parágrafo.

Exemplo: 

        
          <p>Exemplo de tag p</p>

Resultado abaixo:

Exemplo de tag p

Tag - mark

Usado para destacar uma cor de fundo. 

        
          <p>Este é um texto de exemplo com <mark>destaque</mark>.</p>

Resultado abaixo:

Este é um texto de exemplo com destaque.

Tag - small

Reduz o tamanho da fonte do texto. Sendo utilizado para notas de rodapé, por exemplo. 

        
          <p>Este é um texto de exemplo com <small>texto em tamanho reduzido</small>.</p>

Resultado abaixo:

Texto em tamanho reduzido.

Tag - i

Utilizada para colocar o texto em itálico. 

        
          <p>Este é um texto de exemplo em <i>itálico</i>.</p>

Resultado abaixo:

Texto de exemplo em itálico.

Tag - u

Utilizada para sublinhar o texto. 

        
          <p>Este é um texto de exemplo com <u>sublinhado</u>.</p>

Resultado abaixo:

Texto de exemplo com sublinhado.

Tag - strong

Sua função é destacar o texto com forte importância. 

        
          <p>Este é um texto de exemplo com <strong>destaque forte</strong>.</p>

Resultado abaixo:

Texto de exemplo com destaque forte.

Tag - ol

Criar listas ordenadas (numeradas). 

        
          <ol>
            <li>Item 1</li>
            <li>Item 2</li>
            <li>Item 3</li>
          </ol>

Resultado abaixo:

  1. Item 1
  2. Item 2
  3. Item 3

Tag - ul

Criar listas não ordenadas (com marcadores). 

        
          <ul>
            <li>Item A</li>
            <li>Item B</li>
            <li>Item C</li>
          </ul>

Resultado abaixo:

  • Item A
  • Item B
  • Item C

Tag - li

Utilizada para criar itens de uma lista, seja ordenada ou não ordenada. 

        
          <ul>
            <li>Item 1</li>
            <li>Item 2</li>
            <li>Item 3</li>
          </ul>
        
        
          <ol>
            <li>Item A</li>
            <li>Item B</li>
            <li>Item C</li>
          </ol>

Resultado abaixo:

  • Item 1
  • Item 2
  • Item 3
  1. Item A
  2. Item B
  3. Item C

Tag - a

Utilizada para criar links (âncoras) para outras páginas ou recursos. 

        
          <a href="https://www.exemplo.com">Link de exemplo</a>

Resultado abaixo:

Link de exemplo

Tag - hr

Utilizada para criar uma linha horizontal que divide o conteúdo. 

        
          <p>Texto acima</p>
          <hr>
          <p>Texto abaixo</p>

Resultado abaixo:

Texto acima

Texto abaixo

Tag - sub

Utilizada para colocar o texto em subscrito (abaixo da linha de base). 

        
          <p>Texto <sub>subscrito</sub>.</p>

Resultado abaixo:

Texto subscrito.

Tag - sup

Utilizada para colocar o texto em sobrescrito (acima da linha de base). 

        
          <p>Texto <sup>sobrescrito</sup>.</p>

Resultado abaixo:

Texto sobrescrito.

Tag - blockquote

Utilizada para citar um trecho de texto de outra fonte. 

        
          <blockquote>
            <p>Este é um exemplo de citação.</p>
          </blockquote>

Resultado abaixo:

Este é um exemplo de citação.
